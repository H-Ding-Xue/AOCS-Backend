## Description

AOCS Service Framework Starter Kit based on [Nest](https://github.com/nestjs/nest).

Refer to the [documentation](docs/intro.md) for detailed explanation on how to use this template. 

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Generator

```bash
# generate module, controller, service and entity
nest g resource <NAME>

# generate module
nest g mo <NAME>

# generate controller and unit test script
nest g co <NAME>

# generate service
nest g s <NAME>
```

## Docker

```bash
# build docker image from root
docker build -t aocs-svc-frame .
```
